What is a Catalog Item?

Catalog items represent applications or services that receive patches. Each catalog item can have multiple associated patches.

  • Item ID: Internal unique identifier for this catalog item
  • Name: Human-readable name for the application
  • Title ID: NUS identifier for the title
  • Latest Version: Current version number
